# 参天AI - API接口总览

## 项目信息
- **项目名称：** 参天AI八字命理分析系统
- **版本：** 1.0.0
- **基础URL：** `http://localhost:8088`
- **接口总数：** 80+

## 核心功能模块

### 1. 认证与授权（4个接口）
- 短信验证码登录/注册
- Token刷新机制
- 验证码有效期查询

### 2. 八字分析（2个接口）
- MCP工具列表
- 格式化八字分析（自动保存历史）

### 3. 趋势分析 🆕（1个接口）
- 大运/流年/流月完整分析
- 重要节点识别
- 风险预警
- AI深度解读

### 4. 多语言支持 🆕（3个接口）
- 支持10种语言
- 翻译分析结果
- 生成多语言报告

### 5. 塔罗占卜（8个接口）
- 完整的塔罗牌数据库
- 多种牌阵解读
- 智能搜索和推荐

### 6. 易经八字（9个接口）
- 六爻卦象生成
- 八字命盘分析
- 运势预测
- 命理咨询

### 7. 紫微斗数（7个接口）
- 命盘生成和解读
- 运势分析
- 合婚分析
- 择日功能
- 可视化图表

### 8. 星座运势（5个接口）
- 12星座信息
- 每日运势
- 星座配对

### 9. DeepSeek AI（3个接口）
- 八字报告生成
- 卦象解读
- 命盘解读

### 10. 分析管理（8个接口）
- 历史记录管理
- 收藏功能
- 统计分析
- 报告导出

### 11. 用户八字信息（6个接口）
- 多八字管理
- 默认八字设置

### 12. 反馈系统（3个接口）
- 用户反馈提交
- 反馈查询

### 13. 知识库（9个接口）
- 分类管理

- 全文搜索

### 14. 支付系统（7个接口）

## 快速开始

```bash
# 1. 启动项目
mvn spring-boot:run

# 2. 获取Token
curl -X POST http://localhost:8088/api/auth/sms/send \
  -H "Content-Type: application/json" \
  -d '{"phone":"13800138000"}'

# 3. 测试接口
curl -X POST http://localhost:8088/api/bazi/formatted \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"gender":"male","solarDatetime":"1990-05-15 10:30"}'
```
