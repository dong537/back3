# 塔罗牌翻转动画和今日运势功能

## 功能概述

根据UI设计，实现了塔罗牌页面的顶部翻转卡片功能，用户点击塔罗牌后会有反转动画特效，并展示对应今日的运势。

## 实现的功能

### 1. 塔罗牌翻转卡片组件 (`TarotCardFlip.jsx`)

**位置**: `src-frontend/components/TarotCardFlip.jsx`

**功能**:
- 页面加载时自动获取随机塔罗牌
- 点击卡片触发翻转动画（3D翻转效果）
- 翻转后显示塔罗牌正面（从数据库获取图片）
- 自动加载并显示今日运势

**特性**:
- 卡片背面：显示"点击翻转"提示
- 卡片正面：显示塔罗牌图片（如果有）或符号
- 翻转动画：流畅的3D翻转效果（0.6秒）
- 今日运势：翻转后自动加载并显示

### 2. 后端API接口

#### 获取所有塔罗牌
- **接口**: `GET /api/tarot/cards`
- **功能**: 返回所有78张塔罗牌的基本信息
- **返回**: 包含id、cardId、cardNameCn、cardNameEn、symbol、imageUrl等字段

#### 获取今日运势
- **接口**: `GET /api/tarot/daily-fortune/{cardId}`
- **功能**: 根据卡牌ID和日期生成今日运势
- **特性**: 
  - 同一天同一张牌的结果是稳定的（基于日期+cardId的种子）
  - 自动判断正逆位
  - 返回运势解读、建议等

### 3. 页面集成

**位置**: `src-frontend/pages/TarotPage.jsx`

在页面顶部添加了"今日塔罗"卡片区域，包含：
- 翻转卡片组件
- 今日运势展示区域

## 技术实现

### 翻转动画

使用CSS 3D变换实现：
```css
transform-style: preserve-3d;
backface-visibility: hidden;
transform: rotateY(180deg);
transition: transform 0.6s;
```

### 数据获取

1. **随机塔罗牌**: 从数据库获取所有牌，随机选择一张
2. **图片显示**: 优先使用数据库中的`image_url`字段，如果不存在则显示符号
3. **今日运势**: 根据日期和cardId生成稳定的随机结果

### 正逆位判断

使用日期+cardId作为种子，确保同一天同一张牌的结果一致：
```java
long seed = today.toEpochDay() + cardId;
boolean isReversed = (seed % 2) == 0;
```

## 数据库字段

使用现有的`tb_tarot_card`表，主要字段：
- `card_id`: 牌ID（1-78）
- `card_name_cn`: 中文牌名
- `card_name_en`: 英文牌名
- `image_url`: 牌面图片URL
- `meaning_up`: 正位含义
- `meaning_rev`: 逆位含义
- `interpretation_up`: 正位详细解读
- `interpretation_rev`: 逆位详细解读
- `advice_up`: 正位建议
- `advice_rev`: 逆位建议

## 使用说明

1. 用户打开塔罗牌页面
2. 页面顶部自动显示一张随机塔罗牌（背面）
3. 用户点击卡片
4. 卡片执行翻转动画（0.6秒）
5. 显示塔罗牌正面（图片或符号）
6. 自动加载并显示今日运势
7. 运势内容包括：
   - 日期
   - 牌名和正逆位标识
   - 运势解读
   - 建议

## 样式说明

- 卡片尺寸：2:3比例（标准塔罗牌比例）
- 背面：紫色渐变背景，带"点击翻转"提示
- 正面：白色背景，显示塔罗牌图片
- 运势区域：紫色渐变背景，带边框和阴影效果

## 注意事项

1. **图片加载失败处理**: 如果图片URL无效，自动回退到显示符号
2. **API错误处理**: 如果API调用失败，使用默认的运势文本
3. **数据格式兼容**: 支持多种API响应格式，确保兼容性
4. **性能优化**: 翻转动画使用CSS硬件加速，确保流畅
