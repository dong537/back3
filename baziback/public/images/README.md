# 图片资源目录

## 📸 使用说明

此目录用于存放应用的图片资源。

## 📋 需要添加的图片

### 微信二维码

**文件名**：`wechat-qrcode.png`

**位置**：`public/images/wechat-qrcode.png`

**要求**：
- 格式：PNG（推荐）或 JPG
- 尺寸：建议 300x300px 或更大
- 内容：微信二维码图片

**如何添加**：
1. 将微信二维码图片保存为 `wechat-qrcode.png`
2. 复制到 `public/images/` 目录
3. 确保文件名完全匹配（区分大小写）

## 🔍 图片路径

在代码中使用时，路径为：`/images/wechat-qrcode.png`
